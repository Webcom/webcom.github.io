Muniqué

Muniqué is an app for iOS and Android that enables the user to read newsletters that the user subscribe to. With Muniqué, the user can benefit from an app that is built for nothing but reading news.

User data
Muniqué requests read-only access to the users Gmail account in order to filter mails that are newsletters.
Muniqué does not store any user data coming from Google's API.
Muniqué does store the token for Google's API in order to not have to ask the user again.
